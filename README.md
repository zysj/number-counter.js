# number-counter.js
The plugin for counting number showed by animation

## Browser Version 
Chrome, FireFox, IE10.0 and above

## Guide
<ul>
  <li>If you want to change the effect of animation of numbers, you can change the css style of class named <b>'before-piece'</b>,<b>'hide-piece'</b>,<b>'show-piece'</b> in file named <b>number-counter.js</b></li>
</ul>
